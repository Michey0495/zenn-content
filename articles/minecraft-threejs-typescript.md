## コードの一部を見せるね

地形生成のコード。Perlinノイズで自然な山を作ってる。

```typescript
// Claudeが書いたコード
private getHeight(x: number, z: number): number {
  let height = 0
  let amplitude = 1
  let frequency = 0.02

  for (let i = 0; i < 4; i++) {
    height += this.noise.noise2D(x * frequency, z * frequency) * amplitude
    amplitude *= 0.5
    frequency *= 2
  }

  return Math.floor(this.baseHeight + height * 15)
}
```

モブのAI。状態機械で動いてる。

```typescript
// これもClaudeが書いた
switch (this.aiState) {

    else this.moveTowards(player.position, this.moveSpeed)
```

## 動かし方

試してみたい？簡単だよ。

```bash
git clone https://github.com/Michey0495/minecraft-survival.git
cd minecraft-survival
npm install
npm run dev
```

http://localhost:5173 で遊べる。

操作方法：
- WASD：移動
- マウス：視点
- 左クリック：ブロック破壊 / 攻撃
- 右クリック：ブロック設置
- スペース：ジャンプ
- Wダブルタップ：ダッシュ
